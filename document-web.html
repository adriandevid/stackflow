terminal — setup
$ sudo curl -o- https://raw.githubusercontent.com/adriandevid/stackflow/refs/tags/0.0.3/install.sh | sudo bash
Linux Nativo WSL2 (Ubuntu/Debian)

Guia de Início Rápido

Importante: O sistema é compatível apenas com distribuições Linux.

1

Instale com Sudo

Execute o comando sudo curl mostrado no prompt principal. O modo administrador é obrigatório para configurar as interfaces de rede do cluster.

Voltar ao prompt ↑
2

Inicie como Root

Digite o comando abaixo sempre precedido por sudo:

sudo stackflow start

* Informe os dados do cluster e defina seu login para a interface.

3

Acesse o Sistema

Após a inicialização bem-sucedida, acesse a interface via navegador.

Gerenciamento Híbrido Potente

Tudo o que você precisa para orquestrar seu ambiente local ou de produção Linux.

Provisionamento K3d/K3s

Suba clusters Kubernetes leves instantaneamente. Escolha entre Docker-based (k3d) ou direto no host (k3s).

Híbrido K8s & Docker

Cadastre aplicações Kubernetes complexas ou containers Docker isolados na mesma interface personalizada.

Segurança de Sistema

Execução isolada e segura utilizando privilégios de administrador para manipulação correta do kernel e rede Linux.
